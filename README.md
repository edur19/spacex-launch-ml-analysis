# SpaceX Launch Data Analysis & Machine Learning Prediction

## Overview
This project was developed as the capstone project for the IBM Data Science Professional Certificate.

The objective was to analyze SpaceX launch data and build machine learning models capable of predicting first-stage landing success based on historical mission parameters.

## Project Workflow
The project included:
- Data collection through APIs and web scraping
- Data wrangling and preprocessing
- Exploratory Data Analysis (EDA)
- SQL analysis
- Interactive data visualization
- Machine learning model development

## Technologies Used
- Python
- Pandas
- NumPy
- SQL
- Scikit-learn
- Plotly Dash
- Jupyter Notebook

## Machine Learning
Several classification models were evaluated to predict launch outcomes and identify the most relevant mission variables influencing booster recovery success.

## Dashboard
An interactive dashboard was developed using Plotly Dash to visualize launch performance, site statistics, payload distribution, and landing success rates.
